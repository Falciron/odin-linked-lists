# odin-recursion
Concise implementation of a [linked list](https://www.theodinproject.com/lessons/ruby-linked-lists), written in Ruby, for [The Odin Project](https://www.theodinproject.com/).

Nearing the end of the Ruby track, this project has taught or reinforced for me:
- A novel data structure, not directly implemented in the Ruby standard library
- Use of instance-referencing variables as 'pointers', when implementing an object-agnostic computer science concept
- Effective techniques to refactor methods and variables to reduce code quantity
- Intentional and early exception-raising to guardrail against invalid consumer calls

## Pre-requisites for Usage

1. Ruby 3+

## Instructions for Usage

1. Type `ruby main.rb` into your terminal to run the provided sample script.

## Rubocop Instructions

1. Type `bundle install` into your terminal to install rubocop.
2. Type `bundle exec rubocop -a` into your terminal to detect issues.
